# Сайты для СТО

Статический лендинг студии «Сайты для СТО».

## Что это за сайт
Сайт студии, которая делает простые коммерческие сайты для небольших автосервисов, частных авто-мастеров, автоэлектриков и диагностов.

## Для кого
- Небольшие СТО
- Частные авто-мастера
- Автоэлектрики и диагносты
- Сервисы по китайским авто

## Локальный запуск
1. Откройте файл `index.html` в браузере.
2. Либо запустите простой локальный сервер:
   - `python -m http.server 8080`
   - откройте `http://localhost:8080`

## Деплой
Проект деплоится как статический сайт в Vercel без сборки.
